ロボットシステム学講義用リポジトリ

## morseコマンド
- 概要
  - 入力されたアルファベットと特定の記号を、モールス信号に変換します。
  - 対応していない文字や記号、空白は全て無視されます。

## 必要なソフトウェア
- Python
    - テスト済みバージョン3.7~3.12

## テスト環境
- Ubuntu 20.04.6 LTS
